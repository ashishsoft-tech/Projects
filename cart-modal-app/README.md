** Add to Cart Task Using Modal

Project Description

This project is a ReactJS application that fetches product data from the
Fake Store API and allows users to add products to a shopping cart using
a modal interface. The application displays products with their image,
title, and price, and provides an Add to Cart functionality.

Users can view all added items in a cart modal and remove items from the
cart if needed. The project uses React components, props, and state
management to handle product and cart data.

The design is responsive and built using Tailwind CSS so that it works
smoothly on both desktop and mobile devices.

------------------------------------------------------------------------

Features

-   Fetch product data from the Fake Store API.
-   Display product image, title, and price.
-   Add products to cart using Add to Cart button.
-   Show alert message if product is already added to cart.
-   Display cart items count in the Navbar.
-   Open a Cart Modal when the cart button is clicked.
-   Display all cart products inside the modal.
-   Remove products from cart using Remove from Cart butto.
-   Cart value updates dynamically.
-   Responsive UI for mobile and desktop.

------------------------------------------------------------------------

Tech Stack

-   ReactJS
-   JavaScript
-   HTML
-   Tailwind CSS

------------------------------------------------------------------------

API Used

Fake Store API\
https://fakestoreapi.com/products

------------------------------------------------------------------------

Conclusion

This project demonstrates a simple shopping cart application built with ReactJS. Users can view products, add them to the cart, see the cart count in the Navbar, and manage items through a modal. The application ensures proper cart handling and provides a responsive and user-friendly interface.
