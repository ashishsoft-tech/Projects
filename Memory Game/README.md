# Memory Game

## Description
A simple Memory Card Game built using HTML, CSS, and JavaScript (DOM manipulation). Players flip cards to find matching pairs.

## Features
- Card flip animation
- Matching logic
- Shuffle cards
- Restart game button
- Win detection
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript (DOM)

## How to Run Locally
1. Download project files
2. Open index.html in browser

## How to Play
1. Click a card to flip
2. Click another card to find match
3. If cards match, they stay open
4. If not, they flip back
5. Match all cards to win
